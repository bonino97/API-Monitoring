Программа обучения
Программа состоит из разных курсов
1 курс
Основы продакт-менеджмента
Что получите по итогам курса:
  • Оцените себя по карте компетенций и поймете вектор собственного развития для того, чтобы добавить успеха в профессии
  • Узнаете особенности профессии и тренды продакт-менеджмента
2 курс
Продуктовые исследования
Что получите по итогам курса:
  • Научитесь проводить исследования рынка, пользователей, конкурентов
  • Научитесь использовать необходимые качественные и количественные исследования
3 курс
Прототипирование и MVP
Что получите по итогам курса:
  • Освоите подход MVP для проверки ценности продукта
  • Создадите готовый прототип и MVP продукта
4 курс
Экономика, метрики и техническая аналитика
Что получите по итогам курса:
  • Научитесь считать экономику продукта
  • Научитесь решать аналитические запросы с помощью SQL
5 курс
Развитие, продвижение, маркетинг
Что получите по итогам курса:
  • Научитесь строить финансовые модели продукта
  • Научитесь создавать медиаплан и составлять P&L продукта
  • Научитесь использовать маркетинговые каналы для продвижения
6 курс
Привлечение инвестиций
Что получите по итогам курса:
  • Поймете, как и где искать инвестиции для продукта
  • Научитесь проводить питч-презентации
  • Составите питч-презентацию для инвестора
7 курс
Карьерный трек
Что получите по итогам курса:
  • Составите идеальное резюме, которое выделит вас среди других кандидатов
  • Пройдете симулятор собеседования
  • Поймете, как можно развиваться на позиции продакт-менеджера: как меняются требования и обязанности в зависимости от грейда
